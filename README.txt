
NovaLex Website
===============

Three-page static site designed for GitHub Pages.

Pages
-----
- `index.html` — Home / Overview
- `services.html` — Services & engagement options
- `contact.html` — Contact form

Brand
-----
- Colors: Primary #0B2C52, Gold #D4A017, Teal #1ABC9C, Light #F4F6F8, Dark #2C3E50
- Fonts: Montserrat (headings), Open Sans (body) via Google Fonts

Deploy to GitHub Pages
----------------------
1. Create a repository (e.g., `novalex-site`) and upload the contents of this folder.
2. In GitHub → Settings → Pages → Select **Deploy from branch** (`main` / `/` root).
3. Visit the published URL once the build completes.

Form Handling
-------------
- Replace the placeholder `action` in `contact.html` with your form endpoint (e.g., Formspree).
- Or change to `mailto:hello@novalex.co.za` if you prefer email launchers (not recommended for reliability).
